Project Description

This project is a Music Player GUI Application developed in Java.
It provides a simple and user-friendly interface to play audio files with basic controls like play, pause, and stop.

Features
 Play music
 Pause music
 Stop music
 Load audio files
 Simple and interactive GUI
 
 Technologies Used
  Java (Object oriented programming)
  Java Swing (for GUI)
  Java Sound API

How to Run
  Clone or download the project
  Open in any Java IDE (like IntelliJ or Eclipse)
  Run the main file
